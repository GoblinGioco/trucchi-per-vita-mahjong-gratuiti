# Contributing to Vita Mahjong Solver Tools

Thank you for your interest in improving our open-source board solver utility! We welcome contributions from developers of all skill levels.

## How to Contribute

1. **Fork the Repository**: Create a personal copy of this project on your GitHub account.
2. **Create a Feature Branch**: Use a descriptive name for your branch (e.g., `git checkout -b patch-matrix-logic`).
3. **Commit Your Changes**: Ensure your code follows JavaScript standard styles and includes appropriate comments.
4. **Submit a Pull Request**: Provide a clear description of the enhancements or bug fixes introduced in your PR.

## Code of Conduct

* Respect fellow developers and contributors.
* Keep discussions focused on the optimization of game mechanics and algorithms.
* Do not submit compiled binary files or third-party executable scripts.
